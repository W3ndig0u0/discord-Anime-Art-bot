# Discord Anime Art Bot 🤖🎨

A dedicated Discord bot designed for anime enthusiasts and art collectors. This bot automates the process of fetching high-quality anime artwork, wallpapers, and character illustrations from various sources directly into your Discord server.

## 🌟 Features

*   **Source Integration**: Fetches metadata and images from popular platforms (e.g., AniList, custom scrapers, or art APIs).
*   **Search Commands**: Find specific character art or series-based illustrations using simple slash commands.
*   **Interactive UI**: Uses Discord Embeds and Buttons for a clean, navigable gallery experience within the chat.
*   **Secure & Fast**: Built with modern asynchronous libraries to ensure high performance even in large servers.

## 🛠️ Tech Stack

*   **Language**: Python
*   **Database**: Local running E2
*   **APIs**: AniList GraphQL API & Custom Image Scrapers
*   **Environment**: Docker-ready for easy deployment

## 🚀 Getting Started

### Prerequisites
*   A Discord Bot Token from the [Discord Developer Portal](https://discord.com/developers/applications).
*   Java 17+ or Node.js 18+ (depending on your final build choice).
*   A running PostgreSQL instance.
